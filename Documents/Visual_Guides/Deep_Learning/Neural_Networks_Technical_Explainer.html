1. Network Architecture

A neural network is a stack of layers. Each layer applies a linear transformation (z = Wx + b) followed by a non-linear activation function. Data flows left-to-right during the forward pass; gradients flow right-to-left during backpropagation.

Key Insight

Each neuron computes a = activation(W·x + b). Without the activation function, stacking layers is pointless — W₂(W₁x) = Ax collapses to a single linear transformation. Activations break this linearity, letting each layer carve the data into increasingly separable regions.

2. Activation Functions

Each activation function has a distinct shape, output range, and gradient behavior. The choice directly impacts whether gradients survive the journey back through many layers.

Which Activation Where?

Layer Activation Why
Hidden layersReLUGradient = 1 for positive inputs, fast, no vanishing
Hidden (dying neurons)Leaky ReLUSmall slope for negatives prevents dead neurons
Output — binarySigmoidOutputs probability in (0, 1)
Output — multi-classSoftmaxOutputs probability distribution summing to 1
Output — regressionLinear (none)No squashing — any real number

Gradient Derivations at a Glance

σ'(z) = σ(z) × (1 − σ(z))   |   tanh'(z) = 1 − tanh²(z)   |   ReLU'(z) = 0 or 1

Sigmoid's gradient maxes at 0.25. Through 10 layers: 0.25¹⁰ ≈ 0.000001 — the gradient vanishes. ReLU's gradient is exactly 1 for positive inputs: 1¹⁰ = 1. This single property is why deep networks became trainable.

3. The Training Loop

Training is a cycle: forward pass → compute loss → backward pass (backpropagation) → update weights. Repeat for every batch in every epoch until the loss converges.

The Chain Rule Is Backpropagation

Backprop is just the chain rule applied layer by layer: ∂L/∂W₁ = (∂L/∂a₃)(∂a₃/∂z₃)(∂z₃/∂a₂)(∂a₂/∂z₂)(∂z₂/∂a₁)(∂a₁/∂z₁)(∂z₁/∂W₁). Each ∂aᵢ/∂zᵢ is the activation derivative. If that derivative is small (sigmoid: ≤0.25), the product shrinks exponentially through layers.

4. Loss Over Epochs

A healthy training run shows loss decreasing smoothly. The gap between training and validation loss reveals overfitting. Below is a typical loss curve for a well-tuned network.

Reading the Loss Curve
  • Both curves dropping — the model is learning. Keep training.
  • Training drops, validation plateaus — beginning to overfit. Consider early stopping, dropout, or more data.
  • Validation rises while training drops — overfitting. Stop training at the best checkpoint.
  • Neither curve drops — learning rate may be too small, or the model is too simple.

5. Vanishing Gradients & Solutions

During backpropagation, gradients multiply through each layer. If the activation derivative is small, the gradient shrinks exponentially — early layers stop learning.

Regularization Techniques

  • Dropout — randomly disables neurons (p=0.5 default). Forces redundant features. Approximates averaging 2N sub-networks.
  • Batch Normalization — normalizes activations to mean=0, var=1. Learnable γ, β let the network undo it if unhelpful. Enables larger learning rates.
  • Weight Decay (L2) — adds λ||W||² to loss. Penalizes large weights, keeps model simple.
  • Early Stopping — stop training when validation loss stops improving. Save the best checkpoint.
  • Data Augmentation — artificially expand training data (flips, rotations, crops). More data = less overfitting.

Initialization Rules

Xavier: W ~ N(0, 2/(n_in + n_out))  → sigmoid/tanh   |   He: W ~ N(0, 2/n_in)  → ReLU

Xavier keeps variance stable for activations centered at zero. He doubles the variance to compensate for ReLU zeroing out the negative half of activations.

Interview One-Liners

  • Why activations? — Without them, any depth collapses to a single linear layer. They add non-linearity so the network can learn curves and regions.
  • Vanishing gradients? — Sigmoid's max gradient is 0.25. Through 10 layers: 0.25¹⁰ ≈ 10⁻⁶. Early layers freeze. ReLU fixes it with gradient = 1.
  • Why ReLU? — No vanishing gradient, sparse activation, and 6× faster than sigmoid (just a comparison vs exponential).
  • Batch norm? — Normalizes each layer's inputs to mean=0, var=1. Stabilizes training, allows larger learning rates, mild regularization.
  • Dropout? — Randomly disables 50% of neurons per batch. Forces independent features. At test time, approximates averaging 2N sub-networks.
