## FAANG Agentic AI Interview Guide

### Interview Rounds (Typical Structure: 5-7 Rounds)

**Round 1: Recruiter Screen (30-45 min)**
- Background discussion, role fit, salary expectations
- Basic understanding of AI agents and your experience

**Round 2: Technical Phone Screen (45-60 min)**
- Coding problem (LeetCode medium/hard)
- Basic ML/LLM concepts
- System design light discussion

**Round 3-4: Technical Deep Dive (2 rounds, 45-60 min each)**
- LLM fundamentals, prompt engineering, RAG architectures
- Agent frameworks (LangChain, LangGraph, AutoGen, CrewAI)
- Tool use, function calling, memory systems

**Round 5: System Design (60 min)**
- Design an agentic system end-to-end
- Scalability, reliability, cost optimization

**Round 6: Behavioral/Leadership (45-60 min)**
- STAR format questions
- Collaboration, ambiguity handling, impact stories

**Round 7: Bar Raiser/Hiring Manager (45-60 min)**
- Culture fit, long-term vision, team dynamics

---

### Example Interview Questions by Category

#### LLM & Agent Fundamentals
1. "Explain the difference between ReAct, Plan-and-Execute, and Tree-of-Thoughts agent architectures. When would you use each?"
2. "How do you handle hallucinations in an agentic system that makes real-world API calls?"
3. "Design a memory system for a long-running agent. How do you balance context window limits with conversation continuity?"
4. "What's the difference between fine-tuning, RAG, and prompt engineering? When would you choose each?"

#### Coding & Implementation
5. "Implement a basic ReAct agent loop with tool calling from scratch (no frameworks)."
6. "Write code to implement semantic chunking for a RAG pipeline."
7. "Implement a retry mechanism with exponential backoff for unreliable tool calls."
8. "Code a simple vector similarity search without using external libraries."

#### System Design
9. "Design an AI agent that can autonomously debug production issues, access logs, run queries, and propose fixes."
10. "Design a multi-agent system for customer support that handles escalations, knowledge retrieval, and human handoff."
11. "How would you build an agentic code review system that integrates with GitHub and provides actionable feedback?"
12. "Design a cost-efficient agentic pipeline that processes 1M documents daily."

#### RAG & Retrieval
13. "Your RAG system returns irrelevant results 30% of the time. How do you diagnose and fix this?"
14. "Explain hybrid search. When would you combine keyword and semantic search?"
15. "How do you evaluate RAG quality? What metrics would you track?"

#### Safety & Reliability
16. "How do you prevent prompt injection attacks in an agent that executes user-provided instructions?"
17. "Design guardrails for an agent with access to production databases."
18. "How do you implement human-in-the-loop for high-stakes agent decisions?"

#### Behavioral (Amazon LP / Google Googliness style)
19. "Tell me about a time you shipped an ML system that failed in production. What did you learn?"
20. "Describe a situation where you had to make a technical decision with incomplete information."
21. "How do you prioritize when building AI features with uncertain outcomes?"
