1. Logistic Regression — Full Pipeline

Why the gradient is (ŷ−y)·x

This elegant result comes from the chain rule through sigmoid + BCE loss. When ŷ > y (overconfident), the gradient is positive → weights decrease. When ŷ < y (underconfident), gradient is negative → weights increase. The model self-corrects proportionally to the error.

2. Decision Tree — Splitting Math

Weighted Gini = (6/10)×0.278 + (4/10)×0.375 = 0.167 + 0.150 = 0.317
Gini Gain = 0.48 − 0.317 = 0.163 ← choose the split with highest gain
Entropy Alternative

Entropy uses −Σ pᵢ log₂(pᵢ) instead of 1 − Σ pᵢ². Information Gain = Entropy(parent) − weighted Entropy(children). Both metrics pick similar splits in practice. Gini is faster to compute (no logarithm), which is why scikit-learn uses it by default.

3. Random Forest — Under the Hood

Why two sources of randomness?

Bootstrap alone isn't enough — if one feature dominates (e.g., income), every tree will split on it first, making them correlated. Feature subsampling forces diversity. Correlated trees don't reduce variance when averaged. Decorrelated trees do. That's the whole insight behind Random Forest.

4. Gradient Boosting & XGBoost

Key XGBoost Hyperparameters
  • n_estimators — number of boosting rounds (100–1000). More = better fit, risk of overfitting.
  • learning_rate (η) — shrinkage per round (0.01–0.3). Lower = more rounds needed but better generalization.
  • max_depth — tree depth (3–10). Deeper = more complex. Start with 6.
  • reg_lambda (L2) — weight regularization. Higher = simpler trees.
  • subsample — fraction of rows per tree (0.5–1.0). Lower = more randomness.
  • colsample_bytree — fraction of features per tree (0.5–1.0). Like Random Forest's feature subsampling.

5. K-Means — Step by Step

K-Means Limitations
  • Assumes spherical, equally-sized clusters — fails on elongated or irregular shapes
  • Sensitive to initialization — use K-means++ (default in scikit-learn) for smarter starting centroids
  • Must specify K in advance — use elbow method or silhouette score to guide the choice
  • Sensitive to outliers — a single outlier can pull a centroid far from the real cluster center
